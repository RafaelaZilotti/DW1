Exercicio 3


Quais são as 3 principais formas de selecionar elementos HTML no CSS? Forneça exemplos com código.


1- ESPECIFICAR A TAG UTILIZADA

ex:

H1{ text-align: center }


2- UTILIZAR SELETOR CLASSE

ex:

.nome_classe{ text-align: center }

Aplicando o seletor classe:

<p class="nome_classe">texto</p>


3- UTILIZAR SELETOR ID

ex:

#nome_id{ text-align: center }

Aplicando o seletor id:

<p id="nome_id">texto</p>
