Tags de código

<code>

A tag <code> é utilizada para definir um código de computador, utilizando a fonte mono-espaçada padrão do navegador

exemplo:
<code>color</code>muda a cor to dexto
A tag color muda a cor to dexto

<samp>

A tag <samp> é utilizada para demonstrar o output de um programa de computador, utilizando a fonte mono-espaçada padrão do navegador

exemplo:
Meu computador apareceu a mensagem: <samp>File not found.</samp>
Meu computador apareceu a mensagem: File not found.

<pre>

A tag <pre> é utilizada para definir um texto pre-formatado, utilizando uma fonte de largura fixa e o texto preserva os espaços e as quebras de linha

exemplo:
Isto esta pré-formatado: <pre>Texto<pre>
Isto esta pré-formatado: 

texto
